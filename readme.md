## 规则
一. 中奖次数为2次上限，第3次再中奖则无效重新抽。
二. 中奖不要奖项的当场说明弃权，然后再次抽奖。【注：中奖弃权的也算一次中奖】
三. 参与人员范围：梦葱汉化组全体。
四. 参加抽奖的请把各自的头像图片在**2月19日下午4点30前**发给兔脚。
